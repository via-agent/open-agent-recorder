# Roadmap

## Milestone 1: Recorder foundation

- TypeScript trace schema
- default redaction
- Node.js ingest client
- local HTTP collector
- JSON export

## Milestone 2: Replay

- replay captured model calls
- compare model outputs
- preserve original context and tool inputs
- detect changed behavior between runs

## Milestone 3: Evaluation

- incident taxonomy
- pass/fail evaluators
- latency and cost reports
- provider/model comparison

## Milestone 4: Integrations

- OpenAI SDK adapter
- Vercel AI SDK adapter
- LangChain adapter
- CrewAI adapter
- OpenTelemetry bridge

## Milestone 5: UI

- trace timeline
- tool call inspection
- retrieval context inspection
- replay diff viewer
- privacy review workflow
