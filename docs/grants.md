# Grant Positioning

Open Agent Recorder is positioned as infrastructure for safer production AI systems.

## Why AI companies should care

- Agent failures are expensive and hard to reproduce.
- Replay lets teams compare models before migration.
- Local-first traces reduce vendor lock-in and privacy concerns.
- Redaction improves enterprise adoption.
- Evaluation data can be generated from real incidents instead of artificial benchmarks only.

## Credit use

AI credits are useful for:

- replaying captured incidents across models
- running regression evaluations
- benchmarking tool-call reliability
- testing redaction and safety classifiers
- validating open-weight and closed-model routing strategies

## Initial ask

The first practical credit request should fund API usage for replay and evaluation, not vague experimentation. A clear milestone:

1. Record 1,000 real or synthetic agent traces.
2. Replay them across at least 3 model providers.
3. Publish an open incident taxonomy and benchmark report.
4. Release adapters for popular agent frameworks.
